## 产品背景

### 痛点分析

当前小红书带货文案创作存在以下核心问题：

1. **无标准化流程**：创作者需耗费大量时间摸索平台规则
2. **创作效率低**：不同品类、不同节日的文案风格差异大
3. **转化效果不稳定**：新手难以快速掌握创作逻辑
4. **经验难沉淀**：爆款文案规律无法复用

### 解决方案

通过标准化规则 + 模块化模板，打造文案辅助创作工具：
- ✅ 沉淀平台创作法则（情绪共鸣 + 价值传递 + 精准引流）
- ✅ 提供全品类 / 节日文案模板
- ✅ 降低创作门槛，提升转化效率

## 一、AI文案生成规则

### 1.1 核心创作逻辑

**三大核心**：情绪共鸣 + 价值传递 + 精准引流

**表达方式**：
- 口语化表达
- 场景化描述
- 软性引导

### 1.2 通用写作法则（7步法）

| 步骤 | 核心技巧 | 表达要求 | 示例 |
|-----|---------|---------|------|
| 1. 开头抓眼 | 口语化感叹句式 + 核心亮点 | 每句配1个表情，弱化营销感 | 救命🆘这个素颜神器是来报恩的吧🤣！ |
| 2. 卖点绑定 | 硬价值（功能/性价比）+ 软情绪（场景/体验） | 一句话说清，搭配品类表情 | 百元价位💰，黄黑皮也能放心用！ |
| 3. 痛点刺激 | 突出优惠/性价比 | 通俗话术 + 💰类表情 | 一杯奶茶钱，学生党闭眼冲！ |
| 4. 结构精简 | 三段式（惊喜→卖点→引导） | 每段不超过2句话，多用短句 | - |
| 5. 表情赋能 | 按品类选专属表情 | 每句1-2个，紧跟关键词 | 零食🍪 美妆💄 家居☁️ |
| 6. 标签精准 | 品类词 + 卖点词 + 场景词 | 拒绝无关标签堆砌 | #平价口红 #通勤妆容 #学生党好物 |
| 7. 引导转化 | 分享式语言替代硬广 | 配🚀类表情，传递紧迫感 | 收藏蹲补货/安全给闺蜜/错过拍大腿 |
